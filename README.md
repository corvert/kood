### Devops
- [Server Sorcery 101](https://github.com/corvert/Server-Sorcery)
Manually build, configure, and secure a multi-VM network. This project focuses on core Linux administration, networking, and security practices.

- [Infrastructure Insight](https://github.com/corvert/Infrastructure-Insight)
Develop and containerize a diagnostic application (frontend/backend). Deploy it across your virtual infrastructure, configuring the load balancer to distribute traffic and ensuring all components communicate effectively.

- [Automation Alchemy](https://github.com/corvert/Automation-alchemy)
Write scripts to automate the entire process from the first two projects, creating a pipeline that builds your infrastructure and deploys your app automatically.

- [Sherlock Logs](https://github.com/corvert/sherlock-logs)
Build a monitoring and logging solution on a VM. You'll set up Prometheus, Grafana, and the ELK stack to gain deep visibility into your infrastructure and application, complete with custom dashboards and alerts.

- [Cluster Chronicles](https://github.com/corvert/cluster-chronicles)
Migrate your entire application, CI/CD pipeline, and observability stack from VMs to a local Kubernetes cluster using Minikube. This project involves creating Kubernetes manifests, configuring persistent storage, and managing the application within an orchestrated environment.
